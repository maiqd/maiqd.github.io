## 🛠️ Configuration & Implementation Tutorial

To set up the free DeepSeek V4 integration inside your local Hermes installation, execute the following steps precisely:

1. **Local System Installation:** Prerequisite.
Ensure Hermes Agent is installed locally on your system. Note that Windows support is currently available but running in active beta testing.


2. **Portal Account Creation:** Nous Portal Access.
Navigate to the official **Nous Portal** website. Sign up for a new user account and explicitly select the **Free Tier** configuration option. Keep this browser window active.


3. **Invoke Model Configuration:** Terminal Setup.
Open your terminal or system command prompt. Type the command `hermes model` and press Enter to launch the interactive model selection utility.


4. **Link Portal Authentication:** Select Provider.
From the list of available providers, select Option `1` to select the Nous Portal. This action will launch the portal's agent sign-in sequence, authentication, and handshake protocols to bind your local terminal environment to your free tier subscription.


5. **Target DeepSeek V4 Flash:** Model Activation.
Once authenticated, select **DeepSeek V4 Flash** (which displays as completely free to use). Type the corresponding menu number in the command prompt and hit Enter to set it as your default engine.


6. **Execute Active Environment:** Run Agent.
Launch the functional agent interface by typing `hermes` in your terminal. Verify that the system initializes using the DeepSeek V4 engine.
